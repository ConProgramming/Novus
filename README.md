# electron-boilerplate

# Installation

1. `git clone https://github.com/John-Lin/electron-boilerplate.git`
2. `cd electron-boilerplate`
3. `npm install`

# Build the pages html

```sh
npm run build
```

# Run the app
```sh
npm run start
```
